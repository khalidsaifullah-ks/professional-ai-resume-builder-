# Contributing to Minimalist ResumeBuilder

We welcome and appreciate your contributions! To contribute to this project, please follow these guidelines:

## Code of Conduct

Please maintain a respectful, inclusive, and professional environment in all communications and code reviews.

## How to Contribute

1.  **Fork the Repository:** Create a personal fork of the repository on GitHub.
2.  **Clone the Fork:** Clone the repository to your local development machine.
3.  **Create a Branch:** Create a branch for your feature or bug fix (`git checkout -b feature/amazing-feature`).
4.  **Implement Changes:** Make your modifications in the codebase.
5.  **Validate Code:** Before committing, ensure the TypeScript compiler checks pass cleanly with no errors:
    ```bash
    npx tsc --noEmit
    ```
6.  **Commit Changes:** Write clean, descriptive commit messages.
7.  **Push and Open PR:** Push your branch to GitHub and open a Pull Request against the main branch.

Thank you for helping build Minimalist ResumeBuilder!
